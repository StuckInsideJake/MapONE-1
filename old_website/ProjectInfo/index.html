Project Description

The planetary science community develops, collects, and distributes planetary geologic mapping research. This research includes publications discussing mapping work and the developed map products. Similar to geologic mapping on Earth, planetary maps identify key features such as craters, canyons, and fault lines on other planets. Scientists use these shared resources within the community to map space exploration sites.

The client, the United States Geological Survey (USGS) Planetary Geologic Mapping (PGM) Program, assists the community by developing tools and resources to better access and use planetary data for this purpose. For example, PGM mapped the landing/exploration site for the National Aeronautics and Space Administration (NASA) Mars 2020 Perseverance Rover mission. These maps provide information on the surrounding area to show what missions will encounter.

In response to the client, the project team will create a centralized system for all non-USGS map products. The envisioned product, MapONE, is a user-friendly interface on the USGS website. The interface will display map images, sources, and other related information for a selected region of interest. The interface will connect to one database containing all non-USGS maps. As part of the product, a web scraper will use machine learning techniques to collect and identify new maps in articles and conference papers. Overall, the product will allow researchers to access critical non-USGS map information.

The project team will be using the following technologies to develop MapONE:

  • Python: Has robust libraries that add to the functionality of the language
  • Keras and Tensorflow: Python based machine learning techniques that will be applied to the web scraper portion of the project
  • Flutter: Frontend web based GUI display for backend data
  • Django: Web framework used in combination with PostgreSQL to capture and store metadata to the database
  • PostgreSQL: Backend database languge used to format and store data inside the database
  • Amazon Web Services (AWS): Server service used to host the web based GUI and database
